# 📚 Hướng Dẫn Nhập Từ Vựng Từ File

## 🎯 Tính Năng Mới: Nhập Từ File Word/Excel

Bây giờ bạn có thể nhập từ vựng trực tiếp từ file Word (.docx) hoặc Excel (.xlsx/.xls)!

---

Cột 1 | Cột 2 | Cột 3

## 📁 Cách Tạo File Excel

### **Cách 1: Dùng Microsoft Excel/LibreOffice Calc**

1. Mở Excel
2. Nhập dữ liệu vào 3 cột:
   - **Cột A:** Chữ Hán
   - **Cột B:** Phiên âm
   - **Cột C:** Nghĩa (Tiếng Việt)
3. **Lưu file** với tên: `vocab.xlsx`
4. Sử dụng trong VocabLearner

### **Cách 2: Dùng Google Sheets**

1. Truy cập: https://docs.google.com/spreadsheets
2. Tạo bảng mới
3. Nhập 3 cột dữ liệu
4. Lưu file (File → Download → Excel .xlsx)

### **Cách 3: Dùng File Word (.docx)**

1. Mở Word
2. **Tạo bảng** (Insert → Table)
3. Tạo bảng **3 cột × N hàng**
4. Nhập dữ liệu:
   - Cột 1: Chữ Hán
   - Cột 2: Phiên âm
   - Cột 3: Nghĩa
5. Lưu file (.docx)

## ⚠️ Lưu Ý Quan Trọng

### ✅ **Cách Đúng:**
- ✓ Đúng **3 cột**: Chữ Hán | Phiên âm | Nghĩa
- ✓ Không khoảng trống bất thường
- ✓ Mỗi dòng 1 từ vựng
- ✓ Lưu file dưới định dạng **.xlsx** hoặc **.docx**

### ❌ **Cách Sai:**
- ✗ Chỉ 1-2 cột (thiếu dữ liệu)
- ✗ Hơn 3 cột (quá nhiều dữ liệu)
- ✗ Các dòng trống ở giữa
- ✗ Định dạng **PDF** hay **TXT** (không hỗ trợ)

## 🔧 Xử Lý Sự Cố

### **Vấn đề: File không được nhận**

**Giải pháp:**
- Kiểm tra định dạng file: `.xlsx`, `.xls`, hoặc `.docx`
- Kiểm tra file không bị hỏng
- Thử tạo file mới

### **Vấn đề: Chỉ thêm được 1-2 từ**

**Giải pháp:**
- Kiểm tra file có đủ 3 cột không
- Xóa các dòng trống
- Kiểm tra không có dữ liệu thừa ở cột 4, 5...

### **Vấn đề: Chữ Trung bị lỗi**

**Giải pháp:**
- Kiểm tra file được lưu dưới **UTF-8 encoding**
- Nếu dùng Excel cũ (XLS), thử lưu thành XLSX
- Tạo file mới từ Google Sheets (tự động UTF-8)

## 💡 Mẹo

1. **Sao lưu dữ liệu từ khóa học cũ** bằng cách xuất Excel từ Google Docs
2. **Tạo nhiều file** cho mỗi chủ đề (công nghệ, sản xuất, y tế...)
3. **Import từ từ** - tránh thêm hàng trăm từ cùng một lúc
4. **Sử dụng Google Sheets** để quản lý dữ liệu trước khi import
